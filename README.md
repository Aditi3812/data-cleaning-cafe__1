Data Clean Practice

This is my first data cleaning practice. The notebook `clean.ipynb` contains steps I used to clean `dirty_cafe_sales.csv`.

- Tasks performed: removed duplicates, filled missing values, replaced error strings, normalized Location, converted columns to numeric, and filled numeric NaNs with column means.
- Requirements: `pandas`, `numpy` (install with `pip install pandas numpy`).
- How to run: open `clean.ipynb` in VS Code or Jupyter and run the cells from top to bottom.
